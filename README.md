### Compilation
This project includes a `Makefile` to automate the compilation process. To compile the executable, simply run:

```bash
make
```

This will generate the executable:
```bash
./push_swap
```

### Execution
The program prints a sequence of operations:
``` bash
rra
pb
ra
sa
pa
```
